#### 内存保存指令和数据；CPU解释和执行指令——CPU和内存通过总线连接。

#### ABI： 程序与CPU的接口界面。

#### ABI： Application Binary Interface

- 指令编码
- 指令中的寄存器约定
- 大多数指令可直接访问内存。

#### 每次执行后，EIP自加1（1条指令）

#### 指令的长度不同

#### EIP指令可被CALL，RET，JMP和条件JMP指令修改。
